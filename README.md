# 🎓 스마트 분반 시스템

> 마산대학교 치위생과 | 공정하고 효율적인 학급 자동 배정 프로그램

## 📌 프로젝트 소개

학생들을 여러 반으로 공정하게 자동 배치하는 웹 기반 프로그램입니다.

- ✅ **성적 균등 배분**: 각 반의 평균 성적을 최대한 비슷하게
- ✅ **대표 학생 배치**: 각 반에 대표를 1명씩 균등 배치  
- ✅ **특별 지원 학생 분산**: 느린학습자를 골고루 배치
- ✅ **현재 반 고려**: 같은 반 출신이 한 반에 몰리지 않도록 분산

### 🔐 개인정보 보호

**100% 클라이언트 사이드 프로그램**
- 모든 처리는 사용자의 브라우저 내에서만 실행
- 서버로 데이터 전송 없음
- 개인정보는 사용자의 컴퓨터에만 존재

## 🚀 사용 방법

### 1. demo.html 다운로드
### 2. 더블클릭으로 브라우저 실행
### 3. "🎲 샘플 데이터로 테스트" 또는 파일 업로드
### 4. "🚀 분반 시작" 클릭!
